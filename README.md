#  Meme SupInfo

Un générateur de mèmes, rapide et responsive, conçu en JavaScript pur. Ce projet a été développé pour un rendu académique.

##  Fonctionnalités
- **Édition en temps réel** : Personnalisation instantanée du texte (haut/bas), de la couleur et de la taille.
- **Drag & Drop** : Positionnement libre des textes sur l'image à la souris ou au doigt.
- **Filtres d'image** : Application de filtres dynamiques (Noir & Blanc, Sépia, Flou...).
- **Historique local** : Sauvegarde automatique de vos créations dans le navigateur (LocalStorage).
- **Export HD** : Téléchargement au format PNG haute qualité grâce à `html2canvas`.
- **Partage Social** : Boutons prêts pour partager vos créations.


---

##  Stack Technique
- **Langages** : HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
- **Bibliothèque** : [html2canvas](https://html2canvas.hertzen.com/) pour le rendu d'image.
- **Icônes** : Font Awesome 6.
- **Typographie** :  Google Fonts.
